Come sincronizzare tra due Mac
1

Sul Mac da cui esporti

Premi ⬇ Scarica dati (JSON) — il file ottantunocento-dati.json viene scaricato nella cartella Download.

2

Sposta su iCloud Drive

Nel Finder trascina il file in iCloud Drive → Ottantunocento (crea la cartella la prima volta). iCloud sincronizza automaticamente su tutti i tuoi Mac.

3

Sull'altro Mac

Apri il gestionale, vai su ☁️ Sync iCloud, e trascina il file dalla cartella iCloud Drive nel riquadro a destra. I dati si caricano in 2 secondi.

4

Consiglio: salva spesso

Ogni sera prima di chiudere premi ☁️ Salva su iCloud dalla sidebar. Così il file su iCloud è sempre aggiornato e l'altro Mac è pronto.

📂 Importa Excel

Carica un file .xlsx strutturato come il tuo report. Il sistema riconosce automaticamente i fogli Presenze Personale, Anticipi, Spese Carburante e Saldi Dipendenti.

📊

Clicca per scegliere o trascina qui il file

.xlsx · massimo 10MB

Formato atteso per ogni foglio:
• Presenze Personale: colonne Data · Dipendente · Cantiere · Base (€) · Straord. (€) · Totale (€) · Note
• Anticipi: colonne Data · Dipendente · Importo (€) · Metodo · Note
• Spese Carburante: colonne Data · Prodotto · Litri · Prezzo/L (€) · Totale (€) · Fornitore / Note
• Saldi Dipendenti: colonne Dipendente · Maturato (€) · Anticipi (€) · Saldo (€)
